# Adivina Quién

## ¿De qué trata el juego?

Es una versión en consola del juego Adivina Quién. El programa elige un personaje
secreto al azar y el jugador tiene que adivinar cuál es haciendo preguntas sobre
sus características. Tiene un límite de 10 preguntas por partida.
