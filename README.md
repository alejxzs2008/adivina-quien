# Adivina Quién

## ¿De qué trata el juego?

Es una versión en consola del juego Adivina Quién. El programa elige un personaje
secreto al azar y el jugador tiene que adivinar cuál es haciendo preguntas sobre
sus características. Tiene un límite de 10 preguntas por partida.

## Cómo está organizado el proyecto
El proyecto está dividido en carpetas para que cada cosa tenga su lugar.

La carpeta data guarda los archivos de texto. Ahí está personajes.txt
que tiene la información de todos los personajes, y cuando se juega por
primera vez se crea solo el archivo historial.txt con los resultados.

La carpeta modules tiene todos los archivos de Python menos el principal.
Cada archivo se encarga de una cosa distinta para que el código no quede
todo mezclado en un solo lugar.

La carpeta docs tiene este archivo con la explicación del proyecto.

El archivo main.py está en la raíz de la carpeta y es el que se ejecuta
para iniciar el juego. Ese archivo llama a los demás según los necesita.
